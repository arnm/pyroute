
# pyroute


## Local Development

1. Install python3:

```sh
brew install python
```

2. Install pipenv:

```sh
brew install pipenv
```

3. In project top-level directory, install all packages:

```sh
pipenv install
```

4. Run the script:

```sh
pipenv run python pyroute.py route input/tableinputfile1FIX.txt input/addressinputfile1FIX.txt
```
